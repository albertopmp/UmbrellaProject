Umbre??a Purpose

The purpose of this project was to get hands-on experience with AWS Services. As I was learning to become AWS Cloud Practitioner Certified, I wanted to get my hands dirty and apply all the things I'd been learning during the preparation of the certificate.

I saw that many people have taken the Cloud Resume Challenge as a way to advance their cloud career. Nevertheless, I wanted to do something different and develop a project of my own, solving along the way an issue that I'd been facing since I was just a little boy: always forgetting my umbrella when it rains!
